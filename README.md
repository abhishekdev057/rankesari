# Ran Kesari Camping — Luxury Desert Hospitality

A premium Next.js website for **Ran Kesari Camping**, a Rajasthan-born luxury
tent infrastructure brand specialising in Swiss cottages, royal desert camps,
wedding tent cities and end-to-end event hospitality.

> "Temporary luxury. Permanent impression."

---

## Quick start

```bash
npm install
npm run dev
# → http://localhost:3000
```

Production build:

```bash
npm run build
npm start
```

## Adding your images

The site uses graceful SVG placeholders until you drop in real photos.
See **[`public/brand/README.md`](./public/brand/README.md)** for the
complete manifest — the 9 brand photos + logo, their exact filenames,
and every slot they appear in. Want to swap which photo goes where? Edit
[`lib/brand-images.ts`](./lib/brand-images.ts).

## Brand & Style Guide

### Colour palette
| Token              | Hex       | Use                                   |
|--------------------|-----------|---------------------------------------|
| `ink` / `ink-900`  | `#0a0908` | Primary background (matte black)      |
| `ink-800`          | `#121110` | Surface / cards                       |
| `ink-700`          | `#1a1916` | Hover surface                         |
| `gold` (DEFAULT)   | `#c8a25b` | Primary accent                        |
| `gold-100`         | `#f3e7b8` | Bright shine highlight                |
| `gold-500`         | `#a87f1f` | Deep gold                             |
| `sand` (DEFAULT)   | `#d9c4a0` | Secondary warm tone                   |
| `ivory`            | `#f5efe2` | Primary text on dark                  |

The gold accent shifts between `#8a6618 → #d9b54d → #f3e7b8` in a metallic
gradient (`.gold-text`, `bg-gold-shine`) so it reads as metal, not paint.

### Typography
- **Display:** Cormorant Garamond — section headlines, hero, package names.
- **Serif:** Playfair Display — long form serif body where used.
- **Sans:** Inter — UI text, navigation, captions, forms.
- **Tracking:** `tracking-widest2` (0.32em) on all uppercase eyebrows / buttons.

### Type scale (homepage)
| Token   | Size           | Use                       |
|---------|----------------|---------------------------|
| Hero    | 44 → 92 px     | H1 only                   |
| Section | 40 → 64 px     | Section H2                |
| Card    | 24 → 28 px     | Card titles               |
| Body L  | 16 → 18 px     | Long form                 |
| Body S  | 13 → 14 px     | Card descriptions, meta   |
| Eyebrow | 11 px / 0.32em | Uppercase labels          |

### Component primitives
- `.btn-gold` — Primary CTA (gold border, shimmer sweep on hover)
- `.btn-ghost` — Secondary CTA (ivory hairline border)
- `.card-luxe` — Bordered card with `bg-ink-800/60` glass
- `.glass` — Glassmorphism panel
- `.eyebrow` — Centered eyebrow text with hairline rules
- `.gold-text` — Animated metallic-gold gradient text
- `.divider-gold` — Centered 96px gold divider
- `.vignette` — Cinematic radial vignette for hero sections

### Motion system
- **Reveal:** 30px upward + fade, `cubic-bezier(0.22, 1, 0.36, 1)`, 900ms.
- **Hero parallax:** image scales 1 → 1.12, text drifts 200px on scroll out.
- **Hover lifts:** image zoom 1.06, 1200ms, smooth ease-out.
- **Smooth scroll:** Lenis, easing curve `1.001 - 2^(-10t)`, duration 1.15.
